## 2) no_continuous

**list의 연속된 값을 제거해주는 함수**

> list slice를 사용.
```
    ex1) a[1:3] == 1 ~ 2, a[1:] == 1 ~, a[:3] == ~ 2, a[:] == all
    ex2) a[1:10:2] == 1 ~ 9 까지 2 만큼 증가
    ex3) a[-1] == list의 맨 뒤, a[-1:]  == list의 맨 뒤부터(추가), a[:-2] == list 맨 뒤부터 2개 제외
```

> ex_list.append("ex")를 사용. ex_list라는 list에 ex라는 원소를 추가하는 함수
